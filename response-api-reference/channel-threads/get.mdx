---
title: "Get Channel Threads"
api: "GET https://api.stateset.com/v1/channel_threads"
description: "This endpoint gets a channel threads."
---

### Body

<ParamField body="id" type="string">
  This is the id of the channel thread.
</ParamField>

### Response

<ResponseField name="id" type="string">
  This is the id of the channel_thread
</ResponseField>

<ResponseField name="name" type="string">
  This is the name of the channel_thread
</ResponseField>

<ResponseField name="uuid" type="string">
  This is the uuid of the channel_thread
</ResponseField>

<ResponseField name="user_id" type="string">
  This is the user_id of the channel_thread
</ResponseField>
