---
title: "Get Embeddings"
api: "GET https://api.stateset.com/v1/embeddings"
description: "This endpoint gets all embeddings."
---

### Body

### Response

<ResponseField name="id" type="string">
  This is the ID of the embedding.
</ResponseField>

<ResponseField name="knowledge" type="string">
 This is the text of the embedding.
</ResponseField>

<ResponseField body="metadata" type="string">
  This is the metadata of the embedding.
</ResponseField>

<ResponseField name="user_id" type="string">
 This is the user id of the embedding.
</ResponseField>



<RequestExample>

```bash cURL
curl --location --request GET 'https://api.stateset.com/v1/embeddings' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \

```

</RequestExample>
