---
title: "Get Packing List"
api: "GET https://api.stateset.com/api/packing_list"
description: "This endpoint gets or creates a new packing list."
---

### Body

<ParamField body="id" type="string">
  This is the unique identifier for the packing list. If this is not provided, a new packing list will be created.
</ParamField>


### Response

<ResponseField body="id" type="string">
  This is the unique identifier for the packing list.
</ResponseField>

<ResponseField body="number" type="string">
  This is the number associated with the packing list.
</ResponseField>

<ResponseField body="deliveryDate" type="string">
  This is the date of delivery for the packing list.
</ResponseField>

<ResponseField body="invoice_number" type="string">
  This is the invoice number associated with the packing list.
</ResponseField>

<ResponseField body="arrivalDate" type="string">
  This is the date of arrival for the packing list.
</ResponseField>

<ResponseField body="purchase_order_number" type="string">
  This is the purchase order number associated with the packing list.
</ResponseField>

<ResponseField body="ship_per" type="string">
  This is the shipping information or carrier for the packing list.
</ResponseField>

<ResponseField body="tracking_number" type="string">
  This is the tracking number associated with the shipment for the packing list.
</ResponseField>
