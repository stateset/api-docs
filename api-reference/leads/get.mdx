---
title: "Retrieve Lead"
api: "GET https://api.stateset.com/v1/leads/:id/retrieve"
description: "This endpoint retrieves a lead"
---

### Body

<ParamField body="id" name="user_group_id" type="text" required="true">
  The ID of the user group to which the lead belongs.
</ParamField>
