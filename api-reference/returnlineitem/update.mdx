---
title: "Update Return Line Item"
api: "PUT https://api.stateset.com/api/return_line_item"
description: "This endpoint updates an existing return line item."
---

### Body

<ParamField body="id" type="string">
  This is the id of the return line item to update.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>
