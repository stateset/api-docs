---
title: "Update Warranty Line Item"
api: "PUT https://api.stateset.com/api/warranty_line_item"
description: "This endpoint updates an existing warranty line item."
---

### Body

<ParamField body="id" type="string">
  This is the name of the warranty line item to updated
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.stateset.com/api/warranty_line_item' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \

    "_id": "0901f083-aa1c-43c5-af5c-0a9d2fc64e30",
}'
```

```graphQL GraphQL

               mutation (
                $id: uuid
                $warranty_line_item: warranty_line_items_set_input!
              ) {
                update_warranty_line_items (
                where: { id : { _eq: $id }}
                _set: $warranty_line_item
                ) {
                  returning {
                    id
                    sku
                    name
                    price
                    tax_refunded
                    condition
                    amount
                    flat_rate_shipping
                  }
                }
              }`
```
