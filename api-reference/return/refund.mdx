---
title: "Refund Return"
api: "POST https://api.stateset.com/api/return/:id/refund"
description: "This endpoint refunds an existing return."
---

### Body

<ParamField body="return_id" type="string">
  The ID provided in the data tab may be used to identify the return
</ParamField>

### Response

<ResponseField name="id" type="string">
  The ID provided in the data tab may be used to identify the return
</ResponseField>

<ResponseField name="object" type="string">
  The object type
</ResponseField>

<ResponseField name="success" type="number">
  Indicates whether the call was successful. true if successful, false if not.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request PUT 'https://api.stateset.com/api/return/:id/refund' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "id": "e0901f083-aa1c-43c5-af5c-0a9d2fc64e30"
}'
```

```js Node.js
const returns = await stateset.returns.refund({
```
