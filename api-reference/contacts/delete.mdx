---
title: "Delete Contact"
api: "DELETE https://api.return.com/v1/contact"
description: "This endpoint deletes an existing contact."
---

### Body

<ParamField body="id" type="string">
  The ID of the contact to delete.
</ParamField>

### Response

<ResponseField name="id" type="string">
  The ID provided in the data tab may be used to identify the case
</ResponseField>

<ResponseField name="object" type="string">
  The object type
</ResponseField>

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request DELETE 'https://api.stateset.com/v1/contact' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
		"id": "co_1NXWPnCo6bFb1KQto6C8OWvE"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": "co_1NXWPnCo6bFb1KQto6C8OWvE",
```

</ResponseExample>
