---
title: "List Contact"
api: "GET https://api.stateset.com/v1/contact/list"
description: "This endpoint list contacts."
---

### Body

<ParamField body="limit" required="true" type="string">
  This is the limit of the contacts.
</ParamField>

<ParamField body="offset" required="true" type="string">
  This is the offset of the contacts.
</ParamField>

<ParamField body="order_direction" required="true" type="string">
  This is the order direction of the contacts.
</ParamField>
