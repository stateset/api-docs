Stateset Cloud

### Getting Started

#### 1. Create a new project

Create a new project in the Stateset Cloud by clicking the "New Project" button in the top right corner of the dashboard.

#### 2. Create a new organization

Create a new organization in the Stateset Cloud by clicking the "New Organization" button in the top right corner of the dashboard.

#### 3. Create a new API key

Create a new API key by clicking the "New API Key" button in the sidebar.
